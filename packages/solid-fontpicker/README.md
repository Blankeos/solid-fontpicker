# solid-fontpicker

A Google font picker component for Solid.

- No dependencies (other than Solid)
- No Google API key required
- 1600+ Google fonts, customisable
- Fast, high quality font previews from pre-generated SVG images
- Optionally autoloads fonts

Forked from https://github.com/Mikk3lRo/vue-fontpicker/ & https://github.com/ae9is/react-fontpicker
